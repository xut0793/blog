# RegExp

正则表达式是匹配模式，要么匹配字符，要么匹配位置。

- 匹配字符
  - 横向模糊匹配
    - 量词（重复）：`{m,n} {m,} {m} ? + *` 
    - 贪婪匹配与惰性匹配`?`
  - 纵向模糊匹配
    - 字符组：`[abc]`
      - 字符组的范围表示用`-`: `[1-9a-zA-Z_]`
      - 字符组排队表示`^`: `[^1-9]`
      - 字符组元字符表示：`\d \D \w \W \s \S .`: digit word space
  - 多模式分支： `|`
- 匹配位置
  - 位置元字符表示：`^ $ \b \B (?=p) (?!p)`
    - `^`: 开始位置
    - `$`: 结束位置
    - `\b`: 单词边界，即`\w`和`\W`之间位置
    - `\B`: 非单词边界
    - `(?=p)`： p匹配之前的位置
    - `(?!p)`: 不是P匹配之前的位置
